API 说明

GET /apaas/api/config - 获取当前配置

POST /apaas/api/config - 更新完整配置（Body: JSON）

POST /apaas/api/config/token - 远程更新 token 和 cookie（Body: { token, cookie }）

POST /apaas/api/logs/:namespace/app_manage/list - 应用管理日志列表

GET /apaas/api/logs/:namespace/app_manage/detail?logID=xxx - 应用管理日志详情

POST /apaas/api/logs/:namespace/login/list - 登录日志列表

GET /apaas/api/logs/:namespace/login/detail?logID=xxx - 登录日志详情

POST /apaas/api/logs/:namespace/data_change/list - 数据变更日志列表

GET /apaas/api/logs/:namespace/data_change/detail?logID=xxx - 数据变更日志详情

POST /apaas/api/logs/:namespace/flow_runtime/list - 流程运行时日志列表

POST /apaas/api/scheduler/start - 启动定时同步

POST /apaas/api/scheduler/stop - 停止定时同步

GET /apaas/api/scheduler/status - 查询定时任务状态
